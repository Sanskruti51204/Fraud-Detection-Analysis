# Fraud-Detection-Analysis
Fraud Detection Analysis using Data Visualization and SQL
## Project Overview
This project analyzes financial transactions to identify potential fraudulent activities.

## Tools Used
- Excel
- SQL
- Data Visualization
- Dashboard Design

## Key Insights
- Identified transaction patterns related to fraud
- Created a dashboard to visualize fraud trends
- Analyzed transaction types and fraud distribution
